# 운영 기준별 밸런스 QA

작성일: 2026-05-27
대상 버전: v0.3.0 내부 콘텐츠 빌드

## 1. 목적

`돌봄지원 심사소`의 반복 플레이 가치를 높이기 위해 새 캠페인에 3종 운영 기준을 추가했다.

- 균형 심사: 기존 기본 압박을 유지하는 표준 모드
- 지원 확대: 한시적 보조금으로 비용과 민원 압박이 완화되는 모드
- 긴축 감사: 예산 감시와 민원 압박이 강화되는 고난도 모드

운영 기준은 시작 예산, 일일 예산 재배정, 지원 비용, 민원/누락 위험 증감에 영향을 준다. 저장 파일, 최종 리포트, JSON/CSV 로그, 자동 밸런스 QA에 운영 기준이 기록된다.

## 3. 해석

균형 심사는 기존 캠페인 압박을 보존한다. 권장 경로는 안정권이지만 예산과 민원이 빡빡하게 남아 `불 켜진 창` 엔딩으로 이어진다.

지원 확대는 초보자 또는 스토리 확인용 모드다. 권장 경로에서 예산과 민원이 안정되어 `신뢰 회복` 엔딩에 가까워진다.

긴축 감사는 상용 버전용 반복 플레이 모드다. 권장 판단만 따라가도 고압력 결과가 나오며, 위험 우선 플레이처럼 선택 기준을 조정해야 안정권에 들어온다.

## 4. 남은 검토

- 실제 사람 플레이에서 지원 확대 모드가 너무 쉬운지 확인
- 긴축 감사 모드가 불공정하게 느껴지지 않도록 새 캠페인 시작 시 운영 기준 브리핑을 추가 완료
- 엔딩 기록 화면에서 운영 기준별 완료 이력과 전체 발견 이력을 함께 표시하도록 확장 완료
