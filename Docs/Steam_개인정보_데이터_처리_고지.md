# 돌봄지원 심사소 개인정보 및 데이터 처리 고지

작성일: 2026-05-28
대상 버전: v0.3.0

## 요약

`돌봄지원 심사소`는 합성 돌봄지원 신청 사례를 다루는 의사결정 시뮬레이션 게임이다. 게임 안의 가족, 사례, 문서, 사건, 엔딩은 실제 개인이나 실제 행정 기록이 아니라 프로젝트용 합성 콘텐츠다.

처음 실행할 때는 합성 사례, 실제 개인정보 미사용, 로컬 저장, 서버 자동 업로드 없음, 지원 번들/로컬 데이터 삭제, 실제 판단 비대체 내용을 요약한 고지를 한 번 확인해야 한다. 같은 내용은 설정의 `크레딧/고지` 화면에서 다시 확인할 수 있다.

## 수집하지 않는 정보

- 실제 이름
- 실제 연락처
- 실제 주소
- 주민등록번호나 식별번호
- 실제 가족 사정
- 실제 상담 기록
- 실제 기관 내부 자료
- 실제 행정 담당자 정보

플레이테스트 설문에도 위 정보를 적지 않도록 안내한다. 피드백은 게임의 난이도, UI 가독성, 결과 납득도, 10달러 가치감, 재플레이 의향에 한정한다.

## 로컬에 생성되는 데이터

게임은 플레이어가 직접 저장 또는 내보내기를 실행할 때 다음 파일을 로컬 PC에 생성할 수 있다.

- 플레이 로그 JSON/CSV
- 특별 사건 로그 CSV
- 플레이테스트 세션 요약 JSON
- 플레이테스트 설문 JSON/Markdown
- HTML 분석 대시보드
- 시스템 진단 JSON/Markdown
- 지원 번들 manifest
- 저장 슬롯 파일

이 파일들은 기본적으로 플레이어 PC의 로컬 저장 위치에 생성된다. 게임 자체가 서버로 업로드하지 않는다.

## 로컬 데이터 삭제

설정 화면의 `데이터 삭제` 버튼은 저장 슬롯 파일, 플레이 로그, 플레이테스트 세션 원본, 지원 번들, 성과/엔딩/캠페인 기록을 삭제한다. 실수 방지를 위해 같은 버튼을 두 번 눌러야 실행되며, 게임 실행 파일, Steam 설치 파일, 화면/음량/접근성 같은 설정값은 유지한다.

## 환경 진단 정보

환경 진단 기능은 문제 재현과 저사양 PC 검증을 위해 CPU, GPU, RAM, 화면 크기, 그래픽 설정, FPS 샘플 같은 기술 정보를 로컬 파일로 저장한다. 이 정보는 게임 실행 안정성을 확인하기 위한 것이며, 실제 신원 확인을 목적으로 하지 않는다.

## 지원 번들

지원 번들은 문제 제보나 플레이테스트 회수를 쉽게 하기 위해 최신 로그, 세션 원본, 설문, 환경 진단, 저장 파일을 한 폴더로 묶는다. 지원 번들은 사용자가 직접 생성하고 직접 전달해야 한다. 생성된 manifest는 로컬 사용자 절대경로가 노출되지 않는지 확인하도록 구성되어 있다.

## 플레이테스트 회수 원칙

외부 플레이테스트를 받을 때는 `playtest_sessions/<sessionId>` 폴더 또는 `support_bundles/<bundleId>` 폴더 단위로 회수한다. 테스터 실명 대신 익명 세션 ID를 사용하고, 실제 개인정보나 민감한 가족 경험을 받지 않는다.

## AI 생성 및 합성 콘텐츠 고지

일부 시각 자료와 텍스트 제작 과정에는 생성형 AI가 보조적으로 사용될 수 있다. 게임 내 사례는 실제 상담 기록이나 실제 정책 심사 자료를 복사한 것이 아니라, 수업 프로젝트와 게임 플레이를 위해 구성한 합성 사례다.

## 실제 판단 비대체 고지

이 게임은 교육 및 시뮬레이션 목적의 창작물이다. 실제 복지, 행정, 법률, 의료, 상담 판단을 대체하지 않는다. 게임의 권장 판단과 결과 리포트는 실제 기관의 의사결정 기준으로 사용해서는 안 된다.

## 출시 전 확인 기준

- 상점 페이지와 스크린샷에 실제 개인정보가 보이지 않는다.
- 공유용 플레이테스트 패킷에 `C:/Users`, `AppData`, `LocalLow` 같은 로컬 사용자 절대경로가 포함되지 않는다.
- 로컬 데이터 삭제 QA에서 2단계 확인, 게임 생성 파일/폴더 삭제, 무관 파일 보존, 로컬 절대경로 미포함이 통과한다.
- 첫 실행 고지 QA에서 확인 전 표시, 확인 후 저장, 메뉴 복귀, 필수 고지 문구, 로컬 절대경로 미포함이 통과한다.
- 외부 테스터 요청 문구에 실제 개인정보, 실제 가족 사정, 실제 행정 경험을 적지 말라는 안내가 포함되어 있다.
- Steamworks 제출 폴더의 `store_page/PRIVACY_AND_DATA_NOTICE_KO.md`가 최신 고지와 일치한다.
