## 자동 스캔 기준

다음 패턴을 정규식으로 검색했다.

- 이메일 주소
- 국내 휴대전화/전화번호 형태
- 주민등록번호 형태
- 구체 주소로 오인될 수 있는 시/구/동/번지/아파트/빌라 키워드
- 전화, 휴대폰, 이메일 명시 키워드
- 로컬 사용자 절대경로 노출 패턴: `C:/Users`, `C:\Users`, `AppData`, `LocalLow`

## 결과

스캔 기준을 설명하는 이 문서의 예시 문자열을 제외하면, 검수 범위의 데이터와 공유용 산출물 본문에서 위 패턴은 발견되지 않았다.

게임 내 사례의 가족명은 합성 캐릭터명으로 사용하며, 실제 주소, 연락처, 식별번호, 기관 담당자 실명은 데이터에 넣지 않는다.

게임 안의 크레딧/고지 화면에도 합성 사례, 실제 개인정보 미사용, 로컬 로그 저장, 서버 자동 업로드 없음, 환경 진단과 지원 번들 데이터 범위, 생성형 AI 활용, 실제 행정·법률·의료·상담 판단 비대체 문구를 명시했다.

2026-05-28 추가 검수에서 공유용 플레이테스트 패킷은 `playtest_sessions/<sessionId>/파일명` 상대경로만 기록하도록 수정했다. `Builds/QA/v0.3.0/playtest_packet` 대상 경로 스캔에서도 `C:/Users`, `C:\Users`, `AppData`, `LocalLow` 노출은 발견되지 않았다.
