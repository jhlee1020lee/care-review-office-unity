# Third Party Fonts

## Paperlogy

- File: `Assets/Resources/Fonts/Paperlogy-6SemiBold.ttf`
- Source: https://freesentation.blog/paperlogyfont
- Version: 1.001
- License: SIL Open Font License
- Use: Primary in-game UI font for Korean and Latin text.

The official Paperlogy font page describes the family as free for commercial use and distributed under the SIL Open Font License. The game embeds the SemiBold weight so the Windows build does not depend on a locally installed font.
